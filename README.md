# Library Database

A library management database built with SQL and HTML.

## Features
- Store and manage book records
- Store and manage member ID information
- Keep track of loans

## Files
- `library.sql` - Database schema and sample data
- `index.html` - Frontend UI for the library database

## How to Run
1. Import `library.sql` into your SQL database
2. Open `index.html` in any web browser
